## viteのプロジェクト作成

コマンドを実行すると、プロジェクト名の設定と、いくつかのテンプレートから一つ選ぶ、今回は他との違いを知るため vanilla

```bash
npm init vite@latest

cd startup
npm install
npm run dev
```
